# Fashion MNIST Classifier

Neural network for classifying Fashion MNIST clothing images using TensorFlow/Keras.

### Model
- Sequential API
- Dense layers
- Adam optimizer
- Sparse Categorical Crossentropy loss

### Process
- Preprocessing & train-test split
- Model training with 'model.fit'
- Evaluation using accuracy/loss metrics
